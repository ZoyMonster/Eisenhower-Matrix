使用指南

这是一个基于艾森豪威尔矩阵的工作管理工具，帮助你用「重要 / 紧急」四象限来规划和复盘每天的工作。

一、四象限说明

  • 重要且紧急：今天必须完成的核心任务和突发紧急事项。
  • 重要但不紧急：战略性工作、自我提升、需要主动推进但不马上到期的任务。
  • 不重要但紧急：打断类工作、他人临时请求、可协商或委派的事项。
  • 不重要且不紧急：低价值、可果断放弃或大幅压缩时间的事情。

二、如何使用

  • 点击四个象限右上角的「+ 添加事项」，录入待办。
  • 勾选左侧复选框，将事项移动到右侧「已完成清单」。
  • 支持拖拽：在象限内拖拽可调整优先级，跨象限拖拽可重新归类任务。
  • 右键某个事项可将其「聚焦」，便于今天主攻的关键任务突出显示。

三、日报 / 周报

  • 点击右上角「⚙️」进入设置，配置 Gemini API Key。
  • 在设置中点击「📄 导出日报.txt」，基于当天已完成事项生成工作日报。
  • 点击「📆 导出周报.txt」，自动汇总最近 7 天的完成记录生成工作周报。

建议你每天结束前，用 3–5 分钟勾选完成事项并导出日报，每周末导出一次周报，持续复盘、稳步进步。
