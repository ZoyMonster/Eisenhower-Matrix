# 打包 DMG 说明

## 步骤

### 1. 安装 Node.js
确保已安装 Node.js（推荐 v16 或更高版本）
```bash
node --version
npm --version
```

### 2. 安装依赖
```bash
npm install
```

### 3. 测试运行（可选）
```bash
npm start
```

### 4. 打包为 DMG
```bash
npm run build:dmg
```

打包完成后，DMG 文件会在 `dist` 目录中，文件名类似：`艾森豪威尔矩阵-1.0.0.dmg`

## 注意事项

1. **首次打包**：electron-builder 会自动下载 Electron 二进制文件，可能需要一些时间
2. **代码签名**：如果需要代码签名，需要在 `package.json` 的 `build.mac` 中添加 `identity` 配置
3. **图标**：如果需要自定义图标，将 `.icns` 文件放在 `build/icon.icns`，并在 `package.json` 中取消注释图标配置

## 安装 DMG

1. 双击 DMG 文件
2. 将应用拖拽到 Applications 文件夹
3. 在 Applications 中找到并运行应用
